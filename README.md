# Stratton Ashworth

## Overview
Stratton Ashworth is a Swift project designed to demonstrate the capabilities of Swift programming. This project serves as a template for building Swift applications and includes a basic structure for source code and tests.

## Installation
To get started with the Stratton Ashworth project, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/StrattonAshworth.git
   ```
2. Navigate to the project directory:
   ```
   cd StrattonAshworth
   ```
3. Build the project using Swift Package Manager:
   ```
   swift build
   ```

## Usage
To run the application, use the following command:
```
swift run
```

## Running Tests
To execute the tests for this project, use:
```
swift test
```
